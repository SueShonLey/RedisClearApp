# Redis多链接定时清空工具
一款支持多链接管理、可立即清空/定期清空Redis数据的实用工具！

## 功能特点
- ✅ 提供**立即清空** Redis数据功能
- ✅ 支持**定期清空** Redis数据（定时任务）
